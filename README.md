Specyfikacja kontrolera:
- ESP32S3 (FLASH 16MB, 8MB PSRAM QSPI),

- INA219 (czujnik poboru prądu).

Obecny kod pozwala przetestować komunikację z kontrolerem i czy pamięć została poprawnie zainicjalizowana.
